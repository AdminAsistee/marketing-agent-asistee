# SEO Intelligence Feature

## Purpose
The SEO Intelligence feature helps content creators identify high-performing keywords, rising search queries, and content ideas using live Google Trends and search interest data. By integrating search trend context directly into the agent-driven blog generator pipeline, this feature ensures that generated blog content is optimized for search engines while providing genuine value to readers.

## How It Works

```
User Keyword & Context
          │
          ▼
   Trend Retrieval (Gemini with Google Search Grounding)
          │
          ▼
   SEO Analysis Agent (Analyzes queries & strategy)
          │
          ▼
   Content Recommendations (Primary/secondary keywords, titles, strategy)
          │
          ▼
    Writer Agent (Writes SEO-optimized, natural content)
```

## Usage Instructions
1. **Open the SEO Page:** Navigate to the **SEO Intelligence** page by clicking the "SEO Intelligence" button in the blog generator header or going to `/seo`.
2. **Enter Keyword:** Fill in your target keyword/topic (e.g., "Pokemon cards") and optional business context (e.g., "TCGNakama marketplace").
3. **Analyze SEO Opportunities:** Click **Analyze SEO Opportunities** to retrieve trend statistics, related queries, rising keywords, and Gemini-curated content ideas.
4. **Send to Generator:** Click **Use in Blog Generator** to save the keywords, strategy, and content recommendations to the local workspace session.
5. **Generate Optimized Content:** The blog generator page will load prefilled with the SEO-oriented specification. Click **Start Content Generation Pipeline** to write a search-optimized, fact-checked blog post.

## Example

### Input
* **Keyword:** `Pokemon cards`
* **Website Context:** `TCGNakama marketplace`

### Output
* **Primary Keyword:** `Pokemon card investing`
* **Secondary Keywords:** `["rare Pokemon cards", "Pokemon card prices", "Pokemon card market trends"]`
* **Suggested Titles:**
  * `Best Pokemon Cards To Invest In 2026`
  * `How To Identify Valuable Pokemon Cards`
* **SEO Content Strategy:** Focusing on card values and market fluctuations, targeting active collectors and hobbyists.

## Technical Overview
* **Next.js & React:** Powered by React client state and local storage integrations to transition keyword context smoothly between pages.
* **TypeScript:** End-to-end type safety for trend data, recommendations, and api routing payloads.
* **Gemini API:** Uses `gemini-3.5-flash` with Google Search grounding to retrieve real-time search trends dynamically and parse suggestions.
* **Supabase Integration:** Captures latency, keywords, and recommendations generated by the SEO analysis, logging transactions to `agent_logs` under the `'seo_agent'` name.

## Future Improvements
